# 静读经藏

一个先从网页开始的佛经阅读 App 原型。

## 打开方式

直接用浏览器打开：

`/Users/bw/sutra-reader/index.html`

独立手机版：

`/Users/bw/sutra-reader/mobile.html`

手机预览页：

`/Users/bw/sutra-reader/mobile-preview.html`

## 第一版功能

- 经文书架：心经、金刚经、佛说阿弥陀经、普门品完整经文
- 咒语书架：准提咒、大悲咒、六字大明咒、往生咒、药师灌顶真言、楞严咒心
- 正文阅读：分段切换、阅读进度
- 拼音辅助：咒语逐句显示普通话拼音，方便跟读
- 用途说明：右侧显示当前经文或咒语传统上常用于什么
- 内容简释：右侧显示当前经文或咒语的大意说明
- 学习详解：左侧有独立详解列表，点击后在主阅读区打开系统化讲解，包含定位、核心思想、结构脉络、读诵方法和日常应用
- 阅读设置：字号、明暗模式、专注模式
- 个人记录：收藏段落、今日札记、完成次数与连续天数
- 本地保存：设置、札记、收藏和进度会保存在当前浏览器里
- 响应式界面：电脑三栏阅读，平板信息区下沉，手机单栏打开并支持横向滑动入口
- 独立手机版：底部导航分为读经、经库、详解、我的，适合后续放进手机 App 壳里
- 离线阅读：发布到 HTTPS 后，首次在线打开会缓存经文、样式和脚本，之后可离线阅读
- 应用图标：支持浏览器书签图标和手机添加到主屏幕图标
